Installation & usage
--------------------

1. Install this package:

    ```bash
    $ composer require --dev antiseptikk/dev-kit
    ```
    
2. Include a configuration file in your `easy-coding-standard.yml`:

    ```yml
    imports:
        - { resource: 'vendor/sylius-labs/coding-standard/easy-coding-standard.yml' }
    ```
